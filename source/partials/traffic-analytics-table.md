|                                             | Counts as Traffic | Counts for Analytics |
|---------------------------------------------|:-----------------:|:--------------------:|
| API Request                                 |        yes        |          no          |
| Automated traffic from bots or load testing |        yes        |          no          |
| Content pre-fetching                        |        yes        |       sometimes      |
